# To-Do CLI App

A command-line utility written in C++ to add and view tasks.

## Features
* Add tasks.
* View tasks you have created.
* Mark tasks as completed.
* Reset your tasks.

## How to run
1. Ensure you have a C++ compiler (like `g++`) installed.
2. Compile the code: `g++ Main.cpp Task.cpp -o ToDo.exe`
3. Run the application: `./ToDo.exe`
